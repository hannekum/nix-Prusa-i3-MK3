# Summary

If you want to relocate your PSU off the frame (outside enclosure, say), you still need a reinforcement brace for the frame that the PSU body provided.  Here is a simple brace that bolts into the same holes.  

You will need 2 M4 hex nuts.  Otherwise uses existing t-nuts and screws.

I recommend print with 30% gyroid infill.
